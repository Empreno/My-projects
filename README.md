# My-projects
1. [Pet_1 (telco)](https://example.com): датасет от телеком компании, задача бинарной классификации: прогнозирование уйдет ли клиент (прекратит пользоваться услугами компании). 
Бизнес применение: снижение оттока клиентов, приоритизация клиентской поддержки, оптимизация тарифов.  
Использован градиентный бустинг над решающими деревьями (Catboost), метрики: Test AUC: 0.8481, Precision: 0.5431,
Recall: 0.7680, Accuracy: 0.7662
