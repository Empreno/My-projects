# My-projects
1. [Pet_1 (telco)](https://github.com/Empreno/My-projects/blob/main/telco.ipynb): датасет от телеком компании, задача бинарной классификации: прогнозирование уйдет ли клиент (прекратит пользоваться услугами компании). Размер 7043 строк    
**•** Бизнес применение: снижение оттока клиентов, приоритизация клиентской поддержки, оптимизация тарифов.  
**•** Для борьбы с дисбалансом классов был использован undersampling (случайное удаление части примеров до достижения баланса классов)  
**•** Использован градиентный бустинг над решающими деревьями Catboost (были рассмотрены другие модели и ансамбли, оптимальный результат с минимальным переобучением у catboost)    
**•** Результирующие метрики: Test AUC: 0.8481, Precision: 0.5431, Recall: 0.7680, Accuracy: 0.7662  
**•** Для дальнейшей корректировки (баланс между precision/recall) требуется уточнение бизнес задачи
